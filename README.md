AWS SDK for elm
===============

[![npm](https://img.shields.io/npm/v/aws-sdk-elm.svg)](https://www.npmjs.com/package/aws-sdk-elm)

This repo contains scripts which read and translate the AWS SDK [apis/*.json][] files into elm packages.
